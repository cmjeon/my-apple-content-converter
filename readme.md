## 애플 아이폰의 이미지파일 heic 동영상파일 mov 을 변환하는 프로그램

heic 를 jpeg 로 변환

mov 를 mp4 로 변환

## 실행방법

v16 이상의 nodejs 설치

프로젝트 루트에 input, output 폴더 생성

패키지 설치

```shell
$ yarn
```

실행

```shell
$ node myContentConverter
```
